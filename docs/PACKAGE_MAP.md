## Import paths

| Consumer | Import |
|----------|--------|
| Go app | `github.com/itcrow/uacryptex/uacryptex` |
| Rust crate | `uacryptex-core = { path = "../crates/uacryptex-core" }` |
| C / cgo | `#include "uacryptex.h"` + link `libuacryptex_ffi.a` |

## Naming convention

All crates, packages, and FFI symbols use the `uacryptex_*` prefix:

- Rust: `uacryptex-core`, `uacryptex-ffi`
- Go: package `uacryptex`
- C: `uacryptex_version`, `UacryptexBuf`, `UacryptexError`
